## Ejercicios Resueltos

De acuerdo a las consignas, nuestro grupo seleccionó y resolvió los siguientes 2 ejercicios:

### Ejercicio 1 - Pila Genérica: 
* **Descripción:** Implementación de una estructura de datos tipo Pila (`Pila<T>`) utilizando un `ArrayList` como almacenamiento interno.
* **Métodos principales:** `apilar()`, `desapilar()`, `cima()`, `estaVacia()` y `tamanio()`.

### Ejercicio 3 - Par<A,B>: 
* **Descripción:** Creación de un repositorio genérico basado en `HashMap<K,V>` para gestionar persistencia en memoria.
* **Métodos principales:** Operaciones de guardado, búsqueda y eliminación.
